UTC AEROSPACE SYSTEMS

Projects in UTC Aerospace Systems, Bangalore 

1.Prognostic Health Management

Description:
Unscheduled maintenance of aircraft components is the most important factor of Aircraft carrier pain index. Rotational components like compressors are the high probable candidates for unscheduled maintenance. Predictive maintenance will thereby reduce the customer pain index.
Tools: Python 3.6, Pandas, NumPy, SciPy and Matplotlib, Matlab/Simulink

Responsibilities:

  • Worked in development of Proof of Concept.
  • Collection of compressor failure data from the airline carriers.
  • Data cleansing and visualization using Python Pandas NumPy and Matplotlib
  • Preparation of diagnostic rule engine for compressor failure
  • Identifying critical parameter causing failure.
  • Study the behavior of parameter variation using simulation models and flight data.
  • Fit a regression model for the parameter variation and do estimate the remaining useful life.
  • Validate the model with the new monthly flight data.
2.System Performance Analysis for Environmental Control Systems (Sep-2014 to Dec 2015)

Description:
Traditionally controller parameter tuning is done for a normal operating condition and using nominal parameter model/system, the effect of adverse operating conditions, manufacturing design parameter variation and end of life parameter variation is not considered.  Monte Carlo techniques and Simulink models will help us studying the controller robustness for parameter and operating condition variations.
Tools: Matlab/Simulink, Parallel Computing, Statistics toolboxes.

Responsibilities:

  • Integration of control (Simulink) and plant model (Simulink / Dymola) in Simulink environment. Development of scripts for running test profiles in matlab.
  • Development of nominal test profiles using Matlab scripts
  • Validation of system performance requirements (time domain and frequency domain) using simulation data.
  • Performing Monte Carlo simulation study (Sobol – Quasi Monte Carlo Sampling) of parameter variations effect on system performance requirements.
  • Performing input profile variation study on system performance requirements.
  • Preparation of engineering report on parameter variation and input variation study.
